## Create a express Application by globally installing the generator ```typescript npm install -g express-generator ``` ## Scaffold a new Express application called myExpressApp by running # go to the new folder and execute npm install: ```typescript express myExpressApp --view pug npm install ``` # From a terminal in the Express application folder ```typescript npm start ``` # The official Javascript node client for communicating with the Kite Connect API. # Pay the Kite API Charges and activate it, use browser to generate the access code for the day ```typescript npm i kiteconnect https://kite.zerodha.com/connect/login?v=3&api_key=XXX ```

## Create a Angular app ```typescript ng new kiteAngular ``` ## Create a new Repo and push the master # go to the new folder and execute npm install: ```typescript git remote remove origin git remote add origin https://github.com/gmanojisaac/AngularZerodha.git git remote -v show git status git log --oneline git add . git commit -am “Setup Github” git push -u origin master ``` # From a terminal in the Angular application folder ```typescript ng build ng serve ``` # The official Javascript node client for communicating with the Kite Connect API. # Pay the Kite API Charges and activate it, use browser to generate the access code for the day ```typescript npm i kiteconnect-ts https://kite.zerodha.com/connect/login?v=3&api_key=XXX ```
